# CSS 單位介紹

### 尺寸

| 絕對單位 | 說明 |
| -------- | ---- |
| in       | 英吋 |
| cm       | 公分 |
| pt       | 磅   |

| 相對單位 | 說明                   |
| -------- | ---------------------- |
| px       | 像素(螢幕上的一個點)   |
| %        | 百分比                 |
| vh       | 置的畫面高度百分比     |
| vw       | 置的畫面寬度百分比     |
| em       | 1em 為當前的字體尺寸。 |

### 顏色


| 單位          | 說明                                   |
| ------------- | -------------------------------------- |
| rgb(x,x,x)    | RGB 值 e.g. rgb(255,0,0)               |
| rgba(x,x,x,x) | RGB 值外加透明度 e.g. rgb(255,0,0,0.8) |
| #rrggbb       | 十六進制數 (比如 #ff0000)              |
